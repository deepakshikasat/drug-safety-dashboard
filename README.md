# Interactive Drug Safety Explorer

    Streamlit dashboard for adverse event profiles and drug safety comparison.

    ## Dataset

    Demo matrix inspired by public FDA FAERS API fields; app code is ready for live openFDA calls.

    ## Methods

    - Adverse event frequency matrix
- Top event summary
- Streamlit exploratory app

    ## Reproduce

    ```bash
    python scripts/run_pipeline.py
    ```

    ## Outputs

    - `data/faers_demo_matrix.csv`
- `outputs/top_adverse_events.csv`
- `figures/top_events.svg`
- `app.py`

    ## Portfolio Note

    This repository uses public or synthetic demonstration data only. It is
    intended to show reproducible computational biology and AI/ML workflow
    design without relying on confidential or employer-owned material.
