Interactive Drug Safety Explorer

Bench to Bytes portfolio analysis report. Public data sources are named explicitly.

Hypothesis

Public FAERS reaction counts can be normalized into searchable drug safety profiles, while preserving the warning that report counts are not incidence.

Data

openFDA drug/event API, queried by medicinal product name for selected public drugs.

Process

  1. Fetched top MedDRA reaction terms for each selected drug.
  2. Built a drug-by-event count matrix.
  3. Summarized top reaction terms and total reports.
  4. Updated the Streamlit app to browse real openFDA-derived counts.

Outputs

  • data/openfda_drug_event_counts.csv
  • data/openfda_event_matrix.csv
  • outputs/top_adverse_events.csv
  • outputs/drug_report_totals.csv

Interpretation

This is now a real public-data product scaffold. The scientific caveat is central: FAERS data are useful for signal exploration, not direct causal or incidence claims.
