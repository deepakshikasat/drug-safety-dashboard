Interactive Drug Safety Explorer

Bench to Bytes portfolio analysis report. Uses only public or synthetic data.

Hypothesis

Public adverse-event reports can be organized into interpretable profiles that help compare drug safety patterns.

Data

Demo adverse-event matrix inspired by public FDA FAERS/openFDA drug event fields; the Streamlit scaffold is ready for live API integration.

Process

  1. Created a drug-by-event frequency matrix.
  2. Summed events across drugs to identify high-frequency adverse event terms.
  3. Built a Streamlit app for drug-level profile review.
  4. Generated a labelled bar chart of top adverse events.

Outputs

  • data/faers_demo_matrix.csv
  • outputs/top_adverse_events.csv
  • figures/top_events.svg
  • app.py

Interpretation

The event profile emphasizes common reporting terms and demonstrates the product pattern: normalize public safety data into a form scientists can search, compare, and discuss.
